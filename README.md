# 🔥 TRUYEN.TECH – Facebook URL to Share Wrapped Converter 🔥

**Công cụ Python chuyển đổi URL Facebook (profile · group · post · page…) sang dạng Wrapped/Share URL.**

**Website chính thức:** [https://TRUYEN.TECH](https://TRUYEN.TECH)

## Tính năng nổi bật

- Chuyển bất kỳ link Facebook nào sang dạng **wrapped URL** (dạng share an toàn, thường dùng trong group, post ẩn danh…)
- Không cần trình duyệt headless, chỉ dùng `requests`
- Hỗ trợ retry thông minh khi request thất bại

## Yêu cầu

- Python 3.8 trở lên
- Thư viện:
  ```bash
  pip install requests urllib3
